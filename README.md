# AI-Driven Hybrid Sorting Framework

## Overview
This project presents an AI-based intelligent sorting system that dynamically selects the most efficient sorting algorithm using Machine Learning.

## Features
- AI-based decision making
- Random Forest model
- Real-world banking dataset
- Data preprocessing and feature engineering
- Performance evaluation with accuracy metrics

## Dataset
Bank Marketing Dataset (Kaggle)

## Model
Random Forest Classifier

## Results
- Accuracy: ~84%
- Balanced classification performance
- Feature importance analysis

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Contribution
This work introduces an adaptive AI-driven approach for intelligent algorithm selection, improving traditional static sorting systems.
